# Lexer_C
A lexer made in C that supports an array of characters and keywords.

## Supported Tokens

### Keywords
- `int`
- `char`
- `for`
- `while`
- `if`
- `else`
- `return`
- `include`

### Characters
- `(`
- `)`
- `[`
- `]`
- `<`
- `>`
- `{`
- `}`
- `;`
- `:`
- `.`
- `,`
- `!`
- `"`
- `#`
- `$`
- `%`
- `&`
- `'`
- `*`
- `+`
- `-`
- `/`
- `?`
- `@`
- `|`
- `~`

### Literals
- Integer literals
- Generic literals (Variable names, function names etc.)
